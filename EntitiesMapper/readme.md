
# EntitiesMapper

Package that allows mapping between classes in a very simple way.





## Features

- Map classes
- Dependency injection
- Copy list asyncronous
- Developed with .NET Core 9


## How to install

To install this nuget package:

```bash
  dotnet add package EntitiesMapper
```
